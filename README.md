# &lt;/&gt; CodePath Academy

A demo landing site for a learn-to-code bootcamp: hero, a 12-week curriculum with click-to-complete progress tracking, and an accordion FAQ. Progress saved in `localStorage`.

**Live demo:** https://skrizidev.github.io/codepath-academy/

## Pages
- **Home** — hero, stats, "what you'll learn"
- **Curriculum** — 7 modules with a live progress bar
- **FAQ** — expandable questions

## Tech
Vanilla HTML / CSS / JavaScript.
